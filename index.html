HTML — veb sahifaning asosi

HTML (HyperText Markup Language) veb sahifaning skeletidir. U orqali matn, rasm, havola, jadval va formalarni belgilaymiz.

  • HTMLga kirish: teg, atribut, element tushunchasi
  • Sahifa strukturası: <html>, <head>, <body>
  • Matn teglari: <h1>–<h6>, <p>, <span>, <br>
  • Havolalar va rasmlar: <a>, <img>
  • Ro'yxatlar: <ul>, <ol>, <li>
  • Jadvallar: <table>, <tr>, <td>, <th>
  • Formalar: <form>, <input>, <select>, <textarea>
  • Semantik teglar: <header>, <nav>, <main>, <section>, <footer>
  • Meta teglar va SEO uchun asosiy belgi qo'yish
Oddiy HTML sahifa misoli
<!DOCTYPE html>
<html lang="uz">
  <head>
    <meta charset="UTF-8">
    <title>Mening birinchi sahifam</title>
  </head>
  <body>
    <h1>Assalomu alaykum!</h1>
    <p>Bu men yaratgan birinchi HTML sahifa.</p>
  </body>
</html>

CSS — dizayn va bezak

CSS yordamida HTML elementlarining tashqi ko'rinishiga ta'sir qilamiz: rang, o'lcham, joylashuv, chegara, fon, animatsiya va hokazo.

  • CSSga kirish: ulantirish usullari (inline, internal, external)
  • Asosiy selektorlar: teg, klass, identifikator, kombinatsiya selektorlar
  • Box model: margin, border, padding, content, box-sizing
  • Ranglar, fon va chegara: background, border-radius, soyalar
  • Shrift va matn: font-family, o'lcham, qatoraro masofa, yuzasini o'zgartirish
  • Maket: display, position, z-index
  • Flexbox asoslari: konteyner va element xususiyatlari
  • CSS Grid asoslari: qator va ustunlar bilan ishlash
  • Responsiv dizayn: @media qoidalari va breakpointlar
  • Holatlar va animatsiya: :hover, transition, @keyframes
Matnni markazga joylash va rang berish
body {
  font-family: "Segoe UI", sans-serif;
  background-color: #f5f5f5;
}

h1 {
  color: #4f46e5;
  text-align: center;
}

JavaScript — sahifaga harakat qo'shish

JavaScript brauzer ichida ishlaydigan til bo'lib, tugmalar, formalar va dinamik kontent orqali foydalanuvchi bilan aloqa qiladi.

  • JavaScriptga kirish: qator, raqam, logik qiymatlar
  • O'zgaruvchilar: let, const, turlar bilan ishlash
  • Ifodalar va shartlar: if, else, switch
  • Tsikllar: for, while, for...of
  • Funksiyalar: oddiy funksiya, strelka funksiyalar
  • Massiv va objectlar bilan ishlash
  • DOM bilan ishlash: element tanlash, matnni o'zgartirish, klass qo'shish
  • Eventlar: klik, forma yuborish, klaviatura eventlari
  • JSON va oddiy API dan ma'lumot olish (fetch)
Tugma bosilganda xabar chiqarish
<button onclick="salomBer()">Meni bosing</button>
<p id="natija"></p>

<script>
  function salomBer() {
    document.getElementById("natija").textContent =
      "Assalomu alaykum, JavaScript ishladi!";
  }
</script>

React — komponentli yondashuv

React — Facebook tomonidan yaratilgan front-end kutubxonasi bo'lib, katta veb-ilovalarni qismlarga (komponentlarga) bo'lib, oson boshqarish imkonini beradi.

  • Reactga kirish: nega React, SPA tushunchasi
  • Sozlash: Create React App yoki Vite orqali loyiha ochish
  • Komponentlar: funksional komponent yaratish
  • Props: komponentga ma'lumot uzatish
  • State: useState hook orqali holat bilan ishlash
  • Effect: useEffect bilan tashqi ta'sirlarni boshqarish
  • Komponentlar o'rtasida ma'lumot ulashish
  • Oddiy router: bir necha sahifaga ajratish tushunchasi
  • Kichik loyiha: to'liq front-end CRUD yoki to-do list

  return <h1>Assalomu alaykum, React!</h1>;
}
